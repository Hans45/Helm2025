# New Factory Presets

Cette banque de patches met en valeur les nouvelles formes d'ondes avancées ajoutées à Helm.

## Advanced Waves

Cette collection de patches démontre les 8 nouvelles formes d'ondes :

### ?? **NW Super Saw Stack**
- **Forme d'onde** : Super Saw (index 19)
- **Type** : Lead synthétique puissant
- **Description** : Utilise la forme d'onde Super Saw avec superposition de 7 dents de scie désaccordées pour créer un son lead massif et large.

### ?? **NW Folded Bell**
- **Forme d'onde** : Folded Sine (index 18)
- **Type** : Texture harmonique complexe
- **Description** : Exploite la sinusoïde repliée pour générer des harmoniques riches avec un caractère percussif de cloche.

### ?? **NW Chirp Sweep**
- **Forme d'onde** : Chirp (index 20)
- **Type** : Effet de balayage fréquentiel
- **Description** : Utilise la forme d'onde Chirp pour créer des balayages fréquentiels dynamiques de 1x à 5x la fréquence fondamentale.

### ? **NW Thin Pulse Bass**
- **Forme d'onde** : Pulse 10% (index 14)
- **Type** : Basse percutante
- **Description** : Exploite l'onde carrée avec un rapport cyclique de 10% pour un son de basse fin et incisif.

### ?? **NW Skewed Pad**
- **Forme d'onde** : Skewed Sine (index 17)
- **Type** : Pad atmosphérique
- **Description** : Utilise la sinusoïde asymétrique pour créer des textures pad uniques avec modulation douce.

### ?? **NW Hybrid Lead**
- **Forme d'onde** : Tri+Square (index 16)
- **Type** : Lead hybride
- **Description** : Combine triangle (70%) et carré (30%) pour un son lead équilibré entre douceur et agressivité.

### ?? **NW Power Chord**
- **Forme d'onde** : Saw+Square (index 15)
- **Type** : Accord puissant
- **Description** : Mélange dent de scie (60%) et carré (40%) pour des accords de puissance saturés.

### ?? **NW Pulse Arp**
- **Forme d'onde** : Pulse 25% (index 13)
- **Type** : Arpège rythmique
- **Description** : Utilise l'onde carrée 25% avec arpégiateur pour des séquences rythmiques énergiques.

## Utilisation

Ces patches sont optimisés pour démontrer les capacités sonores uniques de chaque nouvelle forme d'onde. Ils peuvent servir de base pour vos propres créations en modifiant les paramètres selon vos besoins musicaux.

## Notes techniques

- Tous les patches utilisent les nouveaux indices de formes d'ondes (13-20)
- Modulations optimisées pour mettre en valeur les caractéristiques de chaque forme d'onde
- Effets et filtres configurés pour compléter les harmoniques naturelles
- Compatible avec Helm version 0.9.0 et ultérieure
